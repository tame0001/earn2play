This folder contains circuit design using KiCAD.

# 7-Segment Display
Each segment has 18 LEDs that draw 5mA each.
So, the total current drawn by a segment is 18 * 5mA = 90mA
And each digit will draw 7 * 90mA = 630mA

There will be 4 digits on the display, so the total current drawn will be 4 * 630mA = 2520mA
The power source will be 5V 3A.
